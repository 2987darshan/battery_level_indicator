# BATTERY LEVEL INDICATOR
## Overview
-A simple PCB design for battery level indicator circuit using LED's
## Tools used
-KiCad
## Features
-It gives a battery level indication.
-The LED named FULL glows when the battery is charged 
-The LOW glows when the battery is discharged.
## Limitations
-This is my frist step towards PCB designing and it works well for 9V battery but when the voltage of battery changes or the battery changes,it might not work as expected especially for low voltage cells like a AA battery of 1.5V and for higher voltages like 12V.
## Future refinements
-improving the circuit so that even batteries of different voltages (0-25V) can be tested with the same circuit.(not very accurate).
